window下配置hexo并且部署到github上

#首先下载node.js和git
##安装node.js
浏览器打开nodejs.org点击下载安装时要选择配置环境变量path
在cmd中输入
-node -v
查看是否成功安装nodejs
#安装git
打开git-scm.com/download/win
选择合适的版本下载即可
新版git安装时可以选择直接添加git bash profile到cmd中
并且添加git的PATH
安装完成后在命令行查看git -version查看是否成功安装git
#安装hexo
在cmd中输入
-npm install hexo-cli -g
建立一个blog文件夹（里面装着所有hexo的信息）
-mkdir blog
-cd blog
