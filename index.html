window下配置hexo并且部署到github上

首先下载node.js和git

安装nodejs

浏览器打开 nodejs.org 点击下载安装时

要选择配置环境变量path，

在cmd中输入

  • node -v #查看是否成功安装nodejs

安装git

  1. 打开 git-scm.com/download/win 选择合适的版本下载即可
  2. 新版git安装时可以选择直接添加git bash profile到cmd中,并且添加git的PATH
  3. 安装完成后在命令行查看
    • git -version #查看是否成功安装git

安装hexo

  1. 在cmd中输入


    - npm install hexo-cli -g

  2. 建立一个blog文件夹（里面装着所有hexo的信息）

    • mkdir blog
    • cd blog
    • hexo init #初始化一个hexo（等待时间比较长）
    • hexo s #查看本地中的博客
浏览器输入对应网址（localhost:4000）即可显示本地的hexo页面

将hexo配置到github

  1. 注册一个github账号
  2. 点击右侧头像选择your profile
  3. 点击加号选择new repository 建立一个新的仓库
     <br>***仓库的名字必须是    用户名.github.io
  4. 点击回车
  5. 复制网址

配置hexo环境

在文件夹中进入到blog，修改_config.yml
在最后加上

type: git

repo: 复制的网址

branch: main

    注意每个冒号后面必须有一个空格

修改完成后保存并退出

配置git

- git config --global user.name" 用户名 "
- git config --global user.email" 注册GitHub使用的邮箱 "

在cmd中执行

  • hexo d # 将hexo置入到github仓库

注意：要一直在网页中的登录GitHub，执行时会自动登录账号与密码

刷新仓库网页，在网页地址栏输入 用户名.github.io即可查看博客

改变主题：https://github.com/litten/hexo-theme-yilia
